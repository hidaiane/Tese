Análise de Correlação entre Variáveis de Qualidade da Água

Introdução

Este tutorial ensina a calcular e visualizar coeficientes de correlação de Spearman entre parâmetros de qualidade da água, precipitação e vazão, separados por períodos chuvoso e seco. O passo a passo é detalhado para permitir a reprodução com outros conjuntos de dados.

4. Análises de Correlação

Agora que temos os tipos corretos de variáveis, podemos gerar os gráficos de correlação

4.1 Período Chuvoso

4.1.1 Filtrar dados e renomear colunas, se necessário

dados_chuvoso <- dados %>%
  filter(Estacao != "IP-97",
         Periodo_real == "Chuvoso") %>%
  rename("Precipitação" = "Precip_media",
         "Vazão" = "Vazao_media")

4.1.2 Gerar gráfico

pairs.panels(
  dados_chuvoso[,9:22],
  method = "spearman",
  pch = 21,
  bg = c("#0073C2FF", "#EFC000FF")[factor(dados_chuvoso$Periodo_real)],
  hist.col = "lightgray",
  density = TRUE,
  ellipses = FALSE,
  stars = TRUE,
  scale = FALSE,
  cex.cor = 1,
  cex.labels = 1.1
)

4.2 Período Seco

O mesmo procedimento anterior, mas agora filtrando os dados para o período seco

4.2.1 Filtrar dados e renomear colunas, se necessário

dados_seco <- dados %>%
  filter(Estacao != "IP-97",
         Periodo_real == "Seco") %>%
  rename("Precipitação" = "Precip_media",
         "Vazão" = "Vazao_media")

4.2.2 Gerar gráfico

pairs.panels(

5. Considerações Finais

  • Verifique ausências (NA) nas variáveis.
  • Ajuste o intervalo [,9:22] conforme seu banco de dados.
  • Correlação ≠ causalidade.
